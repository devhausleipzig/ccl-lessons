### Mac Script

```
# Homebrew
/usr/bin/ruby -e \
  "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
brew update
brew install git
brew cask install \
  visual-studio-code \
  google-chrome \
  docker
```
